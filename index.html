About

As a web developer, I specialize in creating visually appealing and user-friendly websites. With a strong background in HTML, CSS, and JavaScript, I have the skills to design and develop websites that not only look great but also provide an intuitive and seamless user experience. I am experienced in various web development frameworks and technologies, such as React, Angular, and Node.js. I regularly work with RESTful APIs, including The Movie DB API, Open Trivia API, and PokeAPI, integrating real-time data into dynamic interfaces.

My projects have utilized Firebase extensively for authentication, database management (Firestore), and hosting. I have built responsive, component-based web applications using React, React Router, and TailwindCSS, as well as deployed production-ready builds via Netlify and Firebase Hosting. From streaming service clones like Netflix, Hulu, and Realtor.com to static UI recreations of platforms like Google, Gmail, and Twitter, I focus on crafting polished, functional front-end experiences.

I am also skilled in using Vite for rapid front-end development, Axios for API handling, and tools like Figma and Bootstrap for UI/UX design. My ability to design for both desktop and mobile platforms ensures a consistent experience across all devices. I also have experience with TypeScript, JSON, Google Firebase, AWS, React JS, React Native, Angular JS, UI / UX Design, Figma, Bootstrap, and Google Cloud Platform. Whether you need a simple static website or a complex full-stack web application, I am dedicated to bringing your vision to life and delivering a high-quality, professional online presence.
